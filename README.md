# Code Challenge 03 - Proxy Server API
This is a code challenge for the recruitment process of the company Payever.

## Requirements
* [NodeJS](https://nodejs.org/en/)
* [NestJS](https://nestjs.com/)

## Installation
```bash
# npm
$ npm install
```

## Test
```bash
# npm
$ npm test
```

## Run
```bash
# Development

$ npm start

# Production

$ npm run start:prod
```

## Build
```bash
$ npm run build
```
